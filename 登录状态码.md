# 登录状态码

| 状态     | 状态码   |
| ------ | ----- |
| 未处理    | 10100 |
| 成功     | 10101 |
| 未注册    | 10102 |
| 用户名不存在 | 10103 |
| 密码错误   | 10104 |
|        |       |
